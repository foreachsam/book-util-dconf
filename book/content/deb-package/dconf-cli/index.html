dconf-cli 套件探索

下載套件

下載「dconf-cli」這個套件。

$ apt-get download dconf-cli

解開套件

解開「dconf-cli_0.20.0-1_amd64.deb」

$ dpkg -x dconf-cli_0.20.0-1_amd64.deb dconf-cli

觀看資料夾結構

執行

$ tree dconf-cli
